## Installation

Install my-project with yarn

```bash
  cd my-project
  yarn
```

Generate Prisma
```bash
  npx prisma generate 
```

 Push Schema to DB
```bash
  npx prisma db push 
```

Run the program
```bash
  yarn run dev 
```
